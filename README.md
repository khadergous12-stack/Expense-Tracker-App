# 💰 Expense Tracker App (Data Science Project)

## 📌 Overview

The Expense Tracker App is a data-driven project designed to analyze personal financial transactions and provide meaningful insights into spending behavior. It helps users track expenses, visualize trends, and make informed financial decisions.

---

## 🎯 Problem Statement

Managing personal finances is challenging due to scattered transaction data and lack of insights. This project solves the problem by transforming raw expense data into clear visual analytics.

---

## 💡 Solution

This project:

* Generates synthetic expense data
* Cleans and preprocesses the data
* Performs analysis on spending patterns
* Visualizes insights using charts and dashboards

---

## ⚙️ Features

* 📊 Category-wise spending analysis
* 📈 Monthly income vs expense trends
* 📅 Weekday transaction insights
* 🔥 Heatmap visualization (Month vs Category)
* ⚠️ Overspending detection (Negative savings alert)
* 🎛️ Interactive dashboard using Streamlit

---

## 🛠️ Tech Stack

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Streamlit

---

## 🚀 How to Run

### 1. Clone Repository

```bash
git clone https://github.com/your-username/Expense-Tracker-App.git
cd Expense-Tracker-App
```

### 2. Install Dependencies

```bash
pip install -r requirements.txt
```

### 3. Run Application

```bash
streamlit run app.py
```
